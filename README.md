# Caferecomm

> 가고 싶은 카페의 실시간 정보를 얻을 수 있는 웹사이트

********

### 개발환경

- OS : Windows, Linux
- Development Tool : WebStorm, Eclipse
- Framework : Spring
- Language/Skills : HTML, CSS, Javascript, Java
- DB : MyBatis, Oracle
- Bigdata : Hadoop ecosystem(flume, Kafka, Hbase, Spark)

********
